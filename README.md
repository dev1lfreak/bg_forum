# BoardGames Forum

Веб-приложение для создания и просмотра контента, связанного с настольными играми.

1) Просмотр публикаций
2) Создание контента
3) Редактирование уже выложенных публикаций
4) Сохранение незавершенной публикации
5) Удаление публикации автором и администратором
6) Авторизация
7) Система рекомендаций?
8) Сохранения в закладки публикаций
9) Поиск публикаций по названию/тегам
10) Изменение темы сайта
11) Счётчик просмотров
12) Система оценивания
13) Добавление комментариев к постам
